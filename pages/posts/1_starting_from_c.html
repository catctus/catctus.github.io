Starting from C

12 dec 2023

This is my first blog post. Finally done with this static blog generator that I coded in C.
It's quite a simple program and the code is preetty messy, but it was a great project to introduce me to the language.
Also a good compliment to the two courses I took at udemy, this beginner course, and then this advance course.
My goal was first to create a markup to html compiler/converter, but took it a bit further and made it in to a blog generator that I called bloggen.
My next project will be to implement some standard data types and try to use void pointers to make them generic: linked lists, stack, queues and binary trees.
Then I'll jump on this and build an intepreter language. to be continued..
